# Task 13: Three-Panel Layout & Integration

Assemble the full application layout and connect all pieces.

## Layout

- [ ] Three-panel layout: left sidebar (paper list), center (PDF viewer), right (chat)
- [ ] Right panel is collapsible
- [ ] Header bar with app title and upload button
- [ ] Responsive sizing — panels resize appropriately

## Integration

- [ ] Selecting a paper loads it in the viewer and shows its chat sessions
- [ ] Text selection in viewer populates context in chat input
- [ ] Visual indicator showing selected text will be sent with the message
- [ ] "Clear selection" action to remove context before sending
- [ ] Deleting a paper navigates away from it if currently viewed

## Polish

- [ ] Loading states for PDF rendering and chat responses
- [ ] Error states for failed uploads, API errors
- [ ] Empty states (no papers, no chats, no messages)
